Banksys manufactures Point of Sales (PoS) terminals that are both used in their own network and sold to third parties. Terminals have two processors. One is the security processor, the other runs appplications on embedded Linux. Applications can be written in a number of languages, including Java. In this assignment, I was involved in a team effort to design architectural features including

  • infrastructure services,
  • the language-independent programming model, e.g. the service calling discipline,
  • communication between applications.
My specific responsibility included
  • design of a Java API to the infrastructure,
  • the Java programming model,
  • a development tool chain.
The latter was based on the J9 virtual machine and the WebSphere Device Developer (WSDD) plugin for Eclipse. A proof of concept implementation demonstrated the soundness of the architecture and APIs.
